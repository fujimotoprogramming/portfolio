Sample App - Twitter風のWebアプリ

Twitter風のWEBアプリを製作しました。
短文の投稿機能、フォロー機能、ログイン・ログアウト機能、メールアドレスの認証機能のほか、
直感的に操作できるユーザビリティが特徴です。 制作に使ったフレームワークはRuby on Railsで、製作期間は約1か月です。

最初の読み込みに時間がかかる場合があります。
サンプルの都合上新規アカウント作成は受付を停止しています。

絵画の歴史 - 静的なWebサイト

西洋絵画を時系列順に画像付きで並べ、その変遷をビジュアル的に楽しみながら学ぶため制作しました。
1ページで完結、要素がふわっと描画される表現、レスポンシブデザインなどが特徴です。
自身が歴史や絵画が好きだったことから、描かれた時代が分かりにくい西洋絵画を時系列順に並べてわかりやすく伝えたいと思い制作しました。
制作に使った技術はHTML、SCSS、JavaScriptで、Renderサーバーで公開しました。製作期間は約2日です。今後、要素の追加などアップデートを予定しております。

  • リンク : https://paintings-history.onrender.com
  • Github : https://github.com/fujimotoprogramming/paintings_history
